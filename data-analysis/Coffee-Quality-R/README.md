# Coffee Quality EDA
Using R, this project performs an exploratory data analysis on the coffee quality data that's used in a seperate predictive project.
# Getting Started
## Tools
* R Studio
* R 4.2.2
## Required Packages
*	ggplot2
*	readxl
*	tidyverse
*	magrittr
*	lubridate
*	reshape
*	tidyr
* funModeling
* Hmisc
