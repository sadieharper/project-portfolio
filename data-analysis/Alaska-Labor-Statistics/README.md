# Alaska Labor Statistics Analysis
This project uses data from the Bureau of Labor Statistics to report on population, employment, and labor metrics for Alaska, US. 
# Getting Started
## Tools
* Jupyter Notebook
* Python 3.10.6
## Required Packages
*	pandas
*	numpy
*	matplotlib
*	scipy
